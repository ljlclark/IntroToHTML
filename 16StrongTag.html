The Strong Element

The Strong element means that the content is important. It begins with the <strong> tag and ends with the </strong> tag.

The Strong element usually contains only text. The contents are normally displayed by a browser as bold or darker text. 

  <strong>Some important text.</strong>

When an element has a special meaning, it is called a "Semantic" element. Semantic tags describe the elements purpose and type of content to the browser and the developer.

The person coding the HTML document is often called the "Developer".
