The HTML Comment

The HTML Comment is a comment in the HTML file that does not display in the HTML page. It looks like this <!-- Comment -->. 

  <html lang="en">
  <head>
    <!-- The Title element displays the text in the page Title Bar. -->
    <title>The Head Element</title>
  </head>
  <body>
  </body>
  </html>
