The Document Element

The first and main Element in an HTML Document is the Document Element. It begins with the <html> tag and ends with the </html> tag. 

  <!DOCTYPE html>
  <html>
    
  </html>

There is only one Document Element for each document and it contains all the other elements and text in that document.

Elements and text inside an element are usually indented two spaces to make it easier to see what belongs to that element.
